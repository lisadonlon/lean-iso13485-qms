# Quality Objectives

**Document ID:** QM-03
**Revision:** 1.0
**Effective Date:** [DD/MM/YYYY]

> **Skeleton.** ISO 13485:2016 §5.4.1 requires quality objectives that are measurable and consistent with the quality policy, established at relevant functions and levels. Set targets you can actually measure with the data you have. For a new system it is acceptable to mark some metrics "baseline in Year 1" and set the numeric target once you have data — reserve fixed targets for the things that are non-negotiable.
